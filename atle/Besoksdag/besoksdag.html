Velkommen til Besøksdag på AFR

Jeg skal vise dere litt av hva vi på IT har gjort så langt iløpet av året her på AFR

Vi har lært 3 forskjellige "kodespråk" så langt, disse er HTML,CSS og Javascript

HTML er språket som er innholdet på siden

CSS er språket som "style" sida, med ting som farge, størrelse eller lignende

Javascript er språket som får ting til å skje, for eksempel når du trykker på en knapp, eller har pilen over
